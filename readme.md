Para ver el proyecto acceder mediante CMD a la ruta de una carpeta donde deseemos descargar el proyecto:
- cd <dirección de la carpeta>
- git init
- git clone https://github.com/Rubenpard/ctnnotariado.git


### Parte técnica CSS

Diseño mobile-first, html semántico, responsive con metodología BEM y SASS. 
He utilizado la metodologia SMACSS (Scalable and Modular Architecture) para poder escalar el proyecto generando solo las carpetas necesarias.
También he usado las características de sass como variables, mixins y breakpoint para el responsive.

### Parte técnica JavaScript

He usado normalize('NFD') para descomponer los caracteres acentuados, expresiones regulares para eliminar cualquier carácter que no sea una letra o espacio. Comparación basada en sort(), uso de map y concat.
